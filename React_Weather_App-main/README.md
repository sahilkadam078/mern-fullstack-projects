# 🌤️ React Weather App

A simple, clean and responsive **Weather Application** built with React. Search for any city and get real-time weather information with a beautiful card UI.

Live Link : https://dreamy-pothos-16b309.netlify.app/

## ✨ Features

- Search weather by city name
- Clean and modern weather card display
- Shows current temperature, humidity, wind speed, and weather condition
- Responsive design (works on mobile & desktop)
- Built with pure React + Vite (fast and lightweight)

## 🛠️ Tech Stack

- **React** (with Vite)
- **JavaScript**
- **CSS** (plain CSS – no Tailwind)
- **React Components** (`input.jsx`, `card.jsx`)

## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/K-Pratzz/React_Weather_App.git
   cd React_Weather_App
   ```

2. **Configure API key**
   - Copy `.env.example` to `.env`
   - Set `VITE_OPENWEATHER_API_KEY` to your valid OpenWeather key

3. **Install dependencies and run**
   ```bash
   npm install
   npm run dev
   ```
