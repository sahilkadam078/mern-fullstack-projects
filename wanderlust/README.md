# 🏡 Wanderlust – Airbnb Style Full-Stack Web Application

Wanderlust is a full-stack property listing web application inspired by Airbnb.  
It allows users to create, manage, and review property listings with secure authentication and image uploads.

---

## 🚀 Live Demo
🔗 https://your-railway-link.up.railway.app

---

## 🛠 Tech Stack

### Frontend
- EJS
- Bootstrap / CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### Authentication & Security
- Passport.js (Local Strategy)
- Express Sessions
- connect-flash

### Image Storage
- Cloudinary
- Multer

### Deployment
- Railway

---

## ✨ Features

- User Signup & Login
- Session-based Authentication
- Create, Edit, Delete Listings
- Upload Property Images
- Add Reviews
- Authorization (Only owner can edit/delete)
- RESTful Routing
- MVC Architecture

---

## 🧠 What I Learned

- Implementing authentication & authorization
- Managing sessions and cookies
- Working with cloud image storage
- Structuring backend using MVC
- Handling real-world deployment issues
- Environment variable management
